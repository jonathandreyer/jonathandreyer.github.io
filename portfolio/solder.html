Soldering-iron

Personal project / On going

Goal: Make a low-cost soldering-iron with a tip of high quality. The use of an Arduino board allows to use this module with other components such as an LCD screen. The electronic card has been made to fit in with the Arduino world.

On going: A case is currently being carried out and the prototype is made with a 3D printer. In order to improve the heating, an auto-tuning of PID parameters function is being added to the program.

Partial result: The electronic board is functional. A simple program has been developed and allows the soldering-iron to be used. The board can be used with 7 segments screen. It can be replaced by an additional shield with an LCD screen.

Technologies:

  • Arduino (Open source)
  • Arduino Shield (simplified use)
  • Weller RT tips (high quality tip)
  • Standard power supply (12VDC/5A)
  • 3D Printer (case)
