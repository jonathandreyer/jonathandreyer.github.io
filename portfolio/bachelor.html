Bachelor thesis

HE-Arc / 2015

Goal: Conception of a software that is able to detect certain pieces on a backlight surface and to move them according to their position. This project is also about driving a flexible feeder that moves and returns different pieces.

On going: The development as well as the software programming are currently being made. The project is expected to be finished by the end of August 2015.

Technologies:

  • Matrox Imaging Library (competitive price)
  • C# programming
  • MahApps.Metro (UI metro style)
